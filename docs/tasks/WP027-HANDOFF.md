## Implementation Steps

### 1. Create `scripts/wp/lib/parsers/group-menus-to-slides.ts` [NEW]

Groups `ParsedCustomMenu[]` into `ServiceBlock[]` by walking each menu's nearest `e-con.e-parent` ancestor.

**Signature:**
```typescript
export function groupMenusToSlides(
  $: CheerioAPI,
  menus: ParsedCustomMenu[],
): ServiceBlock[];
```

**Logic:**
- For each menu, find the top-level `e-con.e-parent` ancestor → this is the slide boundary
- Group menus sharing the same parent into one `ServiceBlock`
- Assign `id`: `services-01`, `services-02`, etc. (1-indexed)
- Assign `slide`: `"01/03"`, `"02/03"`, etc. (total = number of unique parents)
- Map each `ParsedCustomMenu` → `ServiceCategory`:
  - `id`: from `slugify(menu.title)`
  - `title`: `menu.title`
  - `eyebrow`: `menu.eyebrow`
  - `image`: `menu.image`
  - `imagePosition`: `menu.imagePosition` (default `"left"`)
  - `rotating`: `menu.rotating`
  - `items`: `menu.items`
- Slide `heading`: first eyebrow or title from the slide's first menu, or fallback to "Services"
- CTA text: configurable, defaults to:
  - Slides 1 & 2: `"If you're facing any technology-related challenges hindering your growth, let's talk."`
  - Last slide: `"If you're sure where or how to start, but refuse to give up, let's talk."`
  - Import these from `src/content/text-bands.ts` via `textBands.servicesCtas`
